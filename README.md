# 🚀 Prodify: Empowering Students Through Innovative Productivity Tools

![Prodify Banner](/public/app-preview-qsvHccVI.png) Prodify adalah ekosistem produktivitas *frontend-first* yang dirancang secara spesifik untuk **Mahasiswa**. Prodify memadukan manajemen waktu, fokus, dan kesehatan mental ke dalam antarmuka *gamification* yang interaktif.

🌐 **Live Demo:** [Kunjungi Website Prodify](https://prodify-logicraft.vercel.app/)

## 📋 Informasi Kompetisi (IFEST 14)
* **Nama Website:** Prodify
* **Nama Tim:** Logicraft
* **Dengan Backend:** Tidak (100% Local Storage & State Management)
* **Target Pengguna Utama:** Mahasiswa / Pelajar

## ✨ 5 Fitur Utama (The Big 5)
Sesuai regulasi maksimal 5 fitur produktivitas, Prodify berfokus pada:

1. **Task & Activity Manager:** Manajemen tugas menggunakan Kuadran Eisenhower untuk membedakan tugas *Urgent* (besok deadline) dan *Important* (Skripsi/Jurnal).
2. **Hyper-Focus Engine:** Pengatur waktu fokus kustom (Pomodoro) dengan visualisasi rileks untuk sesi belajar panjang tanpa gangguan.
3. **Smart Notes:** Editor catatan bersih bebas distraksi untuk merangkum materi kuliah. Dilengkapi fitur *Automasi Template Skripsi*.
4. **Atomic Student Habits:** Pelacak kebiasaan harian (hidrasi, baca jurnal) untuk menjaga fondasi akademik yang sehat.
5. **Dashboard Hub & Cognitive Guard (🔥 INOVASI):** * **Gamification:** Mahasiswa mendapat EXP dan memelihara kebun virtual (Neko Mascot) setiap kali menyelesaikan tugas. 
   * **Cognitive Guard:** Sistem AI Heuristik yang mendeteksi jika jadwal harian terlalu padat (Overload Energy) dan secara otomatis memicu layar *Mental Relaxation* (Teknik pernapasan 4-7-8) untuk mencegah mahasiswa *burnout*.

## 🛠 Teknologi yang Digunakan
* **Framework:** React.js & Vite
* **Styling:** Tailwind CSS
* **Animasi:** Framer Motion (Micro-interactions)
* **Ikon:** Lucide React
* **Penyimpanan:** LocalStorage API
